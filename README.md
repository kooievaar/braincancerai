The missing files in this github repository are the scripts, the model, and the mri datasets. The full repository is hosted here: https://danakajesus.dpdns.org/H%3A/__MRI/braincancerai , it has all the files in the project. This github repo only contains the pdf with the paper, a public key for veryfing the pdf, and in the /scripts branch you can find the script files and the requirement file and the mri data.

Download all the files (research a command for downloading all the files recursively yourself, it's quite a few files)

Run unet_training.py like so:
cd <insert path to scripts here, for example H:/__MRI/braincancerai in my case or /home/<username> on linux and apple>
python ./unet_training.py

this creates the model. or download the model file i prebuilt

further instructions are in the paper pdf file, the official publication file. (how do I put this on arxiv)
